# 📡 Jason's AI 現代新聞情報室 (Modern AI News Hub)

這是一個全自動、具備即時聯網能力的 AI 代理人（Agent）伺服器。本系統會自動攔截 **當天最新的 CNN 國際頭條（且保證每次進入不重複）**，並由駐站的 **Gemma 3 AI 情報官** 進行專業分析與批判。

本系統支援 **A2A (Agent-to-Agent) 協議**。即使不透過瀏覽器，你也可以直接從 VS Code 或其他終端工具發送指令，在同一個回應畫面中獲得完整的「新聞標題、網址、雙方對話、以及 AI 總結」。

---

## 💻 開發者 / 同學 本地端測試指南 (VS Code)

如果你下載了這份專案，並使用 VS Code 開啟，請依照以下步驟直接進入 AI 世界：

### 1. 啟動環境

在 VS Code 的終端機 (Terminal) 中輸入：

```bash
npm install
npm start
```
