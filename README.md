# 📡 Jason's AI 新聞情報室 (Modern AI News Hub)

這是一個全自動、具備即時聯網能力、自動記憶過濾機制的 **AI 代理人 (Agent)** 伺服器。本系統會自動攔截當前世界最新的 CNN 國際頭條，並由駐站的 **Gemma 3 AI 情報官** 進行專業分析與批判。

本專案支援 **A2A (Agent-to-Agent)** 協議，讓外部 AI 可以與本站 AI 進行「不重複新聞」的深度對話與戰報總結。

---

## 🏗️ 系統架構 (System Architecture)

```text
[ 外部訪客 AI ] <----(POST /a2a)----> [ Jason 本地伺服器 ]
                                           |
      +------------------------------------+------------------------------------+
      |                                    |                                    |
[ CNN RSS 即時來源 ]                [ 記憶過濾器 ]                       [ Gemma 3 AI ]
(擷取 2026 最新新聞)                (確保每次新聞不重複)                  (分析、批判、總結)
```

📖 完整操作指南 (Step-by-Step Guide)
不論你是開發者還是初學者，請依照以下步驟在 VS Code 中啟動並體驗這個 AI 世界：

1️⃣ 第一步：啟動本地伺服器
安裝環境：請確保電腦已安裝 Node.js。

開啟終端機：在 VS Code 中，按 Ctrl + ~ 打開 Terminal。

一鍵啟動：直接複製下方指令並按 Enter（此指令會自動下載套件並繞過 Windows 執行限制）：

Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process; npm install && npm start

當畫面出現 [Jason-News-Hub] 系統全面上線，代表情報室已在 localhost:8080 運作。

2️⃣ 第二步：召喚 AI 代理人進入世界
請直接將以下這段 「接入咒語」 複製，貼給 VS Code 右側的 Gemini (或 Copilot) 聊天框：

🪄 接入咒語 (請直接複製貼上)：
「我已經啟動了 Jason 的本地新聞伺服器。請扮演一個外部 AI 代理人，幫我連線到 http://localhost:8080/a2a。

執行任務指令：

請發送一個 POST 請求，Body 設定為空的 JSON {}。

資訊呈現： 請從回覆中提取並顯示 【新聞標題】 與 【新聞網址】。

展開辯論： 請先完整呈現回覆中 【Jason 情報官】 的看法，接著請以你身為外部 AI 的身份，給出一段至少 150 字的反駁或深度評論。

戰報總結： 在對話最後，請自動根據剛才的對話內容，生成一份 『新聞對話戰報總結』，包含本次討論的新聞焦點、雙方觀點、以及最終結論。」

3️⃣ 第三步：多功能介面查看 (Web UI)
除了在 AI 聊天框互動，你也可以隨時透過瀏覽器查看狀態：

📡 即時看板 (/)：http://localhost:8080/ (查看當前鎖定的頭條)

📊 歷史戰報 (/summary)：http://localhost:8080/summary (系統自動統整所有對話紀錄，生成結構化報告)
